# Trace samples

Place **exported Langfuse traces** (JSON) or screenshots here for coursework / demos.

This folder satisfies the rubric expectation of a `/traces` (or `traces`) directory with sample observability artifacts.

## What to add

1. Run a few queries against your app with Langfuse keys set.
2. In the Langfuse UI, open a trace → export or copy the trace JSON if your plan supports export.
3. Save as `sample_trace_<date>.json` (or paste a minimal structure into `sample_trace.json`).

The bundled `sample_trace.json` is a **minimal illustrative shape** only — replace with a real export when submitting.
